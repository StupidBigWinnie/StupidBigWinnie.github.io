# Markdown Syntax Note

## Text

### BOLD
use two * to make bold:
**BOLD is HERE**

### ITALIC
use one * to make italic:
*ITALIC is HERE*

### Link
use bracket to make a link, use parenthesis to write the url link
	[A Link HERE](www.bing.com)
